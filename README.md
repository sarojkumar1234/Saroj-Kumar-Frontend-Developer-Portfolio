# Saroj Kumar - Profile Website

A beautiful, modern profile/portfolio website built with React, TypeScript, Vite, and Tailwind CSS.

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** - Fast build tool
- **Tailwind CSS** - Utility-first styling
- **Outfit** & **DM Sans** - Google Fonts

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Project Structure

```
profile/
├── src/
│   ├── App.tsx      # Main profile page
│   ├── App.css      # Component styles
│   ├── index.css    # Global styles + Tailwind
│   └── main.tsx     # Entry point
├── index.html
└── vite.config.ts
```

## Features

- Dark theme with cyan/teal accents
- Responsive design
